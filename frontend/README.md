# Govtech 2021 TAP Assignment - URL Shortener Frontend

Frontend of URL Shorterner service built using React.

View [live demo](http://govtech-2021-url-shorterner.herokuapp.com/)



## Requirements

NodeJS



## Installation

```
$ cd my-app
$ npm install
```

Ensure that you also have the backend running. Refer to the [README](/tree/main/backend/README.md)  for futher steps.



## Run

Set the backend url as environment variable while starting the app. Remember to include the `/url` endpoint to ur backend url.

```
$ REACT_APP_API_URL=localhost:8000/url npm start
```
